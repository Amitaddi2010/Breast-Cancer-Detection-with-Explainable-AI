🧠 Explainable AI (XAI)

Visual explanations highlighting regions that drove the classification decision.

Generating Grad-CAM…

Grad-CAM uses gradients flowing into the final conv layer to produce a coarse localization heatmap highlighting important regions.

Generating Grad-CAM++…

Grad-CAM++ improves Grad-CAM by using a weighted average of gradients, providing better object localization for multiple targets.

Generating Score-CAM…

Score-CAM is gradient-free and generates explanations by measuring how each activation map affects the model output score.

Generating LIME explanation…

LIME (Local Interpretable Model-agnostic Explanations) segments the image into superpixels and identifies which regions most influence the prediction.

Generating SHAP values…

SHAP (SHapley Additive exPlanations) assigns each pixel a contribution value based on game theory, producing pixel-level attribution maps.
